## 🎨 Design System Applied

### Color Palette
```css
--navy-deep:      #1a365d  /* Primary navy */
--royal-blue:     #2c5aa0  /* Secondary blue */
--orange-warm:    #ed8936  /* CTA buttons */
--teal-accent:    #16B286  /* Checkmarks & accents */
```

### Gradients
```css
/* Purple (Hero sections) */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* Blue (Process, Insights, CTAs) */
background: linear-gradient(135deg, #0A2540 0%, #3A8DDE 100%);

/* Orange-Pink (Incentives banner) */
background: linear-gradient(to right, #f97316 0%, #ec4899 100%);
```

### Design Patterns
- ✅ **Card-Based Layouts** - Modern, scannable content
- ✅ **Glassmorphism** - Semi-transparent elements with backdrop blur
- ✅ **Smooth Animations** - Transform + shadow on hover
- ✅ **Icon-First Design** - Large emojis for visual hierarchy
- ✅ **Responsive Grids** - Auto-fit layouts (1/2/3/4 columns)
- ✅ **Pattern Overlays** - Subtle textures on gradients
- ✅ **Left-Border Accents** - Blue/teal borders on cards
